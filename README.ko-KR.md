# Video Frame Extractor

동영상 프레임 추출용 GUI 툴

Read this in other languages: [English](README.md), [한국어](README.ko-KR.md)

## 시작하기

### 설치

1. [FFmpeg](https://ffmpeg.org/)를 다운로드합니다.
1. FFmpeg를 `ffmpeg` 폴더에 압축해제합니다.

## 실행파일 생성

1. pyinstaller를 설치합니다.
    ```
    pip install pyinstaller
    ```
1. `build.bat`를 실행합니다.
